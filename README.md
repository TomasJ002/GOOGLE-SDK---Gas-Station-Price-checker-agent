/GOOGLE ADK - Gas station prices checker
This agent is responsible for interacting with the user and finding him the cheapest reliable gas station to refuel.
The found gas station is at most a 10 minutes detour from the planned route, and he is querying olny the well known gas stations - OMV, SHELL, Orlen and MOL (Slovnaft in Slovakia)
Due to the fast, dynamically changing gas station prices, the agent will notify you, if the data is older than 3 days, and will offer an alternative with newer fuel price data.
The agent is able to use googleSearch() and googleMaps() tools.
